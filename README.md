WordPressBootstrap
==================

A boilerplate theme for web developers building WordPress themes with Bootstrap and Compass

**Install Node**
http://nodejs.org/

**Install Compass**
http://compass-style.org/install/

**Initialise**
    $ cd ~/path/to/zingdesign-wordpress-bootstrap
    $ [sudo] bash init.sh

**Update**
    $ cd ~/path/to/zingdesign-wordpress-bootstrap
    $ [sudo] bash update.sh
